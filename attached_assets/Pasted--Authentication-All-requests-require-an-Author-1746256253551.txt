=========================
Base URL: https://api.up.com.au/api/v1

=========================
üìò Endpoints & Formats
=========================

1. List Accounts
GET /accounts

2. Retrieve Account
GET /accounts/{account_id}

3. List Transactions
GET /transactions

4. Retrieve Transaction
GET /transactions/{transaction_id}

5. List Transactions by Account
GET /accounts/{account_id}/transactions

6. List Categories
GET /categories

7. Retrieve Category
GET /categories/{category_id}

8. Categorize Transaction
PATCH /transactions/{transaction_id}
Body:
{
  "data": {
    "type": "transactions",
    "id": "{transaction_id}",
    "relationships": {
      "category": {
        "data": {
          "type": "categories",
          "id": "{category_id}"
        }
      }
    }
  }
}

9. List Tags
GET /tags

10. Add Tags to Transaction
POST /transactions/{transaction_id}/relationships/tags
Body:
{
  "data": [
    {
      "type": "tags",
      "id": "tag-name"
    }
  ]
}

11. Remove Tags from Transaction
DELETE /transactions/{transaction_id}/relationships/tags
Body:
{
  "data": [
    {
      "type": "tags",
      "id": "tag-name"
    }
  ]
}
